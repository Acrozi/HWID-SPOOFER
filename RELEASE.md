# HWID Changer v1.0

Windows Hardware ID Changer with graphical interface. Modifies system identifiers to change hardware fingerprints.

## Features

- **Quick Spoofing** - Change all identifiers with one click
- **Selective Options** - DISKS, NETWORK, REGISTRY, HARDWARE, or CUSTOM
- **System ID Viewer** - View all current identifiers and export reports
- **Real-time Console** - Monitor spoofing progress
- **Help Window** - Detailed information and documentation

## What It Changes

**Registry-Based Identifiers:**
- Machine GUID, Product ID, Computer Name
- Volume Serial Numbers (all drives)
- MAC Addresses (all network adapters)
- Monitor/Display IDs, GPU Registry
- Installation IDs, Telemetry IDs

## Requirements

- Windows 10 or Windows 11
- Administrator privileges
- Portable executable - no installation needed

## Installation

1. Download `HWIDSpoofer.exe` from releases
2. Run as administrator
3. No installation required

## Usage

**Quick Start:**
1. Launch `HWIDSpoofer.exe` as administrator
2. Click `SPOOF ALL` to change all identifiers
3. Restart your computer

**Important:** Always restart after spoofing for all changes to apply.

## Important Notes

- Changes require restart to be visible
- May trigger antivirus false positives (normal for registry/system tools)
- Create system restore point before use
- Use at your own risk
